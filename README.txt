Name: Just Book IT

An application to promote reading eBooks among youths. Rewards and badges are given to encourage users to read.
Interest groupss and forums too. 

Design process:

- as a user, I want to be able to connect with other users so a forum and interest groups were included.
- as a user, I want to be able to track what books I read so books I have read are added to my bookshelf
- as a user, I want to be motivated to read books so users get points per book read

Features:
- groups and forum:
allow users to connect with like minded individuals

- bookshelf:
allow users to track what they have read, and what other books are in the collection.
Rewards users with points per book read

-point shop:
allow users to see how many points they have and redeem prizes with points
